# pandora-api

- create venv python3 from IDE or CLI
    `> python3 -m venv venv`
    `> source venv/bin/activate`

- run installation python file for precessing json data for favourite food
    `> python3 install.py`

- run the code locally
    `> uvicorn main:app --host 0.0.0.0 --port 8080 --reload`
